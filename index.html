Tecnologias

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. As tags em HTML nos permite criar elementos na página web, desde mostrar uma imagem ou exibir um simples texto até criar formulários de contato e blocos de elementos.

JavaScript é uma linguagem de programação interpretada estruturada, permitindo que façamos animações e que seja aplicada várias funcionalidades em seu site, está entre uma das linguagens de programação mais utilizadas do mundo.

CSS é um mecanismo para adicionar estilo a um documento web, ou seja, com ele mudamos cores, formas, tamanho de elementos entre muitas outras estilizações da página. Além de ser o responsável por adptações para versões mobile.

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. Fazendo com que as interfaces possam ser muito mais interativas, estilizadas e otimizadas para melhor performance.
